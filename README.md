Tema: Cálculo de volúmenes de corte y relleno
Descripción del problema
En la ingeniería civil, es común que antes de construir una obra se tenga que nivelar el terreno.
Esto implica quitar material (corte) o rellenar donde haga falta (relleno) para alcanzar una altura de diseño.
El cálculo de esos volúmenes suele hacerse a mano o con software especializado, pero este programa lo realiza de forma automática usando Python.
________________________________________
Objetivo
Desarrollar un programa que permita calcular los volúmenes totales de corte y relleno a partir de una matriz de elevaciones del terreno, la altura de diseño y el área representada por cada punto.
De esta manera se aplica la programación para resolver un problema real de la ingeniería civil.
